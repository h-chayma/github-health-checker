Help - Repository Health Checker
Introduction

This tool helps you check the health of a GitHub repository by displaying key metrics like stars, forks, and open issues. You can also view recent activity, including pull requests and commit statistics.

How to Use
  1. Enter a GitHub username in the "GitHub Username" field.
  2. Enter a repository name in the "Repository Name" field.
  3. Click the Check Health button to display the repository data.
Key Features Explained
  • Repository Information: Displays the repository's owner, stars, forks, issues, and basic details.
  • Activity Overview: Includes last commit, total commits, contributors, and how to interpret these metrics.
  • Pull Requests: Shows recent pull requests and their statuses (open or closed).
  • Additional Insights: Describes watchers, languages, and licenses.
